## Front End Starter Template
This is Quick Start Pack for Front End Developer. It's make with Gulp 4

## Features:
1. Gulp.js (v4.0.2)
2. Compile Sass to Css
3. Minify Css and JS
4. Beautify Code
5. Browser Auto Sync
6. Pug JS
